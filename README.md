# 🛡️ Simulador de Phishing

Este proyecto es una aplicación web interactiva que simula ataques de phishing en diferentes canales (correo, WhatsApp y SMS) con el objetivo de educar a los usuarios en ciberseguridad.

## 🚀 Funcionalidades

- Simulación de correos, mensajes y chats
- Identificación de phishing vs mensajes seguros
- Temporizador para simular presión real
- Retroalimentación educativa
- Generación de certificado en PDF
- Simulación de ingeniería social (formulario falso)

## 🎯 Objetivo

Concientizar a los usuarios sobre los riesgos del phishing mediante una experiencia práctica e interactiva.

## 🛠️ Tecnologías usadas

- HTML
- CSS
- JavaScript
- jsPDF
